# Numerical Methods Library

Библиотека численных методов, реализованная на Java 21.

Проект разработан в рамках научной работы
"Разработка библиотеки численных методов на языке Java".

## Описание

Данная библиотека предоставляет набор алгоритмов численного анализа,
которые могут быть использованы в различных программных проектах
для решения математических задач.

Основная цель проекта — создание модульной и расширяемой
библиотеки численных методов.

## Реализованные методы

### Решение нелинейных уравнений

- Метод бисекции
- Метод Ньютона

### Решение систем линейных уравнений

- Метод Гаусса

## Требования

- Java 21
- Maven

## Сборка проекта

```bash
mvn clean install
```
